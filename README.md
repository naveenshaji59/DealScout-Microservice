# DealScout Microservice
An automated, WAF-evading data pipeline and REST API built for real-time e-commerce price tracking.

## Tech Stack

Backend: Python, FastAPI, Uvicorn

Data Pipeline: SQLite3, Selenium, Undetected-Chromedriver

## Core Features

WAF Bypass: Utilizes stealth automation to navigate bot-mitigation firewalls (403 Forbidden evasion).

Dynamic Scraping: Explicit wait architecture to capture JS-loaded elements.

REST API: Fully documented local API serving JSON payloads for frontend integration.
